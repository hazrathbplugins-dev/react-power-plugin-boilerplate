== Description ==

React Powered Plugin is a WordPress plugin that demonstrates how to build custom Gutenberg blocks using React. It provides a foundation for developing modern, interactive blocks for the WordPress block editor.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/react-powered-plugin` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Start building your custom React-powered blocks!

== Frequently Asked Questions ==

= Does this plugin require Gutenberg? =
Yes, this plugin is designed for the WordPress block editor (Gutenberg).

= Can I extend this plugin? =
Absolutely! Use it as a starting point for your own custom blocks.
